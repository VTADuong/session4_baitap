What is CSS

Cascading Style Sheet (CSS) is used to format the layout of a webpage.

Whith CSS, you can control the color, font, the size of text, the spacing between elements, how elements are positioned and laid out, what background colors are to be used, diffrent displays for diffrent devices and screen sizes, and much more!

Tip: The word cascading means that a style applied to a patent element will so apply to all children elements within the parent. So, if you set the color of the body text to "blue," all headings, paragraphs, and other text elements within the body will also get the same color (unless you specify something else)!
